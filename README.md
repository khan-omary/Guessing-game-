# Number Guessing Game 🎯

A fun and interactive **Number Guessing Game** built with **HTML, CSS, and JavaScript**.  
Try to guess the number between 1 and 20 that the computer randomly selects. Perfect for learning JavaScript basics or just having fun!

## How it works
1. Choose a number between 1 and 20
2. The computer randomly selects a number
3. If your number matches the computer's, you win 🎉
4. You have 5 attempts to guess correctly

## Features
- Interactive gameplay
- Score tracking
- Works on desktop and mobile
- Fun and educational

- HTML

- JavaScript
